# 推し旅 日本演出现场决策助手

一个轻量手机 Web App 原型。

## 定位

面向计划去日本看演出的海外粉丝，解决“信息碎片太多，缺少针对具体场馆和个人情况的行动建议”的问题。

当前 MVP 只保留三个模块：

- 场馆实战指南：到场时间、最近车站、排队建议、末班车风险。
- 文化须知：现场礼仪、禁止携带、应援规则、拍照规定。
- 综合建议：根据预算、优先级、是否第一次去和场馆风险，给出“值得去 / 可以等下一次 / 考虑替代方案”。

## 使用方式

直接打开 `index.html` 即可运行。项目不依赖网络、框架、API Key 或后端服务。

## 文件

- `index.html`：手机 App 原型结构
- `styles.css`：暖黑金日系杂志风视觉样式
- `app.js`：离线场馆数据和建议逻辑

## 面试讲解点

这个 Demo 的重点不是做普通旅游攻略，而是展示一个垂直场景的产品洞察：海外粉丝去日本演出时，真正需要的是“针对这场演出、这个场馆、这个人的现场决策建议”。

正式产品可以接入票务、交通、场馆公告、社媒经验帖和 OTA 数据，延展为文化旅游或票务平台的增值能力。
